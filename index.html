Offerings

Programming languages

We prefer using modern, Functional Programming languages like Scala & Clojure. We are experts in traditional OO languages like C#, Java, and love the new featrues in EcmaScript 6.

Databases

We have deep expertise in various RDBMSes (SQL Server, Oracle) and have deployed production applications in Cassandra and Neo4j. We are also evaluating JanusGraph using Apache Tinkerpop.

Front end

We love all the development happening in the JavaScript world, and have authored a book on AngularJS. We love React, and have built network visualizations using Cytoscape.js and D3.js.

Frameworks

We've used Play framework, Asp.Net MVC, ORM frameworks (NHibernate, Entity Framework), IoC containers (Spring.Net, McWire), testing frameworks (JUnit, NUnit, Scala Test, Selenium).

Usability

We specialize in the usability testing using formative and summative assessments. We support use case building, user research, develop interview techniques, conduct cognitive walkthroughs and implementation of IEC62366-1:2015 standard
